# Project.v2

## Установка

```bash
npm install

## Запуск 

npm run dev


Веб-приложение для внутренней HR/админ-панели с модулями:
- авторизация;
- дашборд с аналитическими виджетами;
- сотрудники и карточка сотрудника;
- регионы и филиалы;
- посещаемость;
- документы;
- мастер найма сотрудника.

## Технологии

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- TanStack Query
- i18next (ru/uz)
- local-agro-ui (UI-компоненты)
- Recharts

## Требования

- Node.js 18+ (рекомендуется 20+)
- npm 9+
